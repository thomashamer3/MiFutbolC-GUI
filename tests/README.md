# Pruebas unitarias (Unity minimal)

Este directorio contiene un *runner* de pruebas unitarias estilo Unity y un conjunto de pruebas para `utils`.

## Ejecutar en Windows

1. Compilar:
   - Ejecuta `build_tests.bat` en la raíz del proyecto.
2. Ejecutar:
   - Ejecuta `MiFutbolC_tests.exe`.

## Notas

- Se usa un *subset* minimal de Unity incluido en `tests/unity`.
- Los tests actuales validan funciones sin dependencia de I/O ni base de datos.
